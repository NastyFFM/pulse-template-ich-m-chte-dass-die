# Ich Möchte Dass Die

Ich möchte, dass die Gitarre das ist. Eine einfache POS ist die App, die auf Gitta the published wird und zwar nicht nur als Repo. Das ist das Besondere, sondern auch bei den Github iOS Seiten bei dem User in seinen Apps. Jetzt gibt es ein Problem. natürlich können POS Apps immer Daten speichern, wenn sie unter POS laufen. Aber natürlich nicht, wenn sie bei Gitta iO laufen, deswegen gibt es einen zusätzlichen Schritt. Alle Apps müssen als progressive Web Apps installiert werden können. Alle Apps können werden die User Daten als Cookies speichern und bieten die Möglichkeit an ein Back-up downloaden oder ein Back-up wieder hochzuladen, damit man auf einem anderen Browser weitermachen kann. Das ist eine Standard Funktionalität in diesem Template, dass die Apps so angelegt werden müssen also frontend mit Vanilla, CSS, HTML und Vanilla, Java, Skript und Publish of Gitarre als Repo und als öffentliche App unter github IO
